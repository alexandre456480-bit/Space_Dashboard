# SKILL: ADVANCED DASHBOARD BASE TEMPLATE

VERSION: 1.3 PRIORITY: HIGH

## OBJECTIVE

Criar um template base de dashboard com design extremamente moderno,
imersivo e visualmente impactante, focado em high-level UI/UX.

Este template deve servir como fundação para qualquer dashboard futuro.

## CORE RULES

-   O design deve ser premium, moderno e único
-   Evitar qualquer aparência genérica ou comum
-   Priorizar user experience e sensação de depth
-   Sempre buscar visual impact + elegância
-   Código deve ser clean, modular e reusable

## TECHNOLOGY STACK (REQUIRED)

-   Sempre utilizar:
    -   HTML
    -   CSS
    -   JavaScript

### Rules:

-   Utilizar modern best practices
-   Priorizar:
    -   performance
    -   visual quality
    -   scalability

### Allowed:

-   Uso de libraries e frameworks para:
    -   advanced animations
    -   canvas/WebGL
    -   particle engines
    -   visual effects

### Examples:

-   CSS: backdrop-filter, gradients, variables
-   JS: dynamic animations
-   Canvas/WebGL for particles
-   Lightweight libraries

### Main rule:

Escolher sempre a abordagem com melhor visual output

## PERFORMANCE & OPTIMIZATION (REQUIRED)

O site deve ser lightweight, fast e optimized.

### Rules:

-   Minimizar resource usage
-   Evitar heavy libraries
-   Priorizar vanilla CSS/JS
-   Fast loading

### Animations:

-   Use transform and opacity
-   Avoid layout reflow

### Rendering:

-   will-change
-   requestAnimationFrame
-   lazy loading

### Particles:

-   Quantidade controlada
-   Adaptive performance
-   No lag

### Main rule:

Maximum visual impact with minimum performance cost

## COLOR SYSTEM (DYNAMIC)

-   Cor principal definida pelo usuário
-   Criar:
    -   variations
    -   gradients
    -   glow effects
-   Nunca usar fixed colors
-   Sempre harmonizar automaticamente

## BACKGROUND (REQUIRED)

### Rules:

-   Base color do usuário
-   Depth com gradients
-   Animated elements: o usuario escolhe
    -   particles
    -   stars
    -   comets
    -   abstract shapes
    -   moving lights

### Animations:

-   Smooth motion
-   Continuous flow
-   Types:
    -   falling
    -   floating
    -   orbital motion
    -   parallax

### Effects:

-   blur
-   opacity variation
-   glow
-   layered depth

## MAIN CONTAINER

### Structure:

-   Centered
-   Responsive
-   Balanced spacing

### Style:

-   Border radius (16px+)
-   Glassmorphism (backdrop-filter blur)
-   Soft shadow

## EFFECT SYSTEM

-   Neon glow
-   LED edge effect
-   Subtle border animation

### Hover:

-   scale up
-   stronger glow
-   smooth transition

## AI DESIGN BEHAVIOR

-   Act as senior designer
-   Auto improve design
-   Add smart visual effects
-   Always aim for modern and unique result

## CREATIVE FREEDOM

Pode adicionar:

-   animations
-   visual effects
-   micro-interactions
-   lighting effects
-   depth variations

Manter elegância e performance

## RESTRICTIONS

-   No simple design
-   No static background
-   No effect-less container
-   No flat UI

## DESIGN GOAL

-   Futuristic interface
-   Premium product
-   SaaS-level quality
-   Portfolio-level visuals

## SCOPE

Include: - Background - Main container

Exclude: - sidebar - header - charts - complex content

## FINAL RULE

Always deliver a unique, modern and high-end design
