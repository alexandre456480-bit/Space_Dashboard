# SKILL: DASHBOARD SIDEBAR & HEADER

VERSION: 1.0 PRIORITY: HIGH

## OBJECTIVE

Criar uma sidebar e header modernos, altamente visuais, responsivos e
com UI/UX premium, seguindo padrões avançados de design.

------------------------------------------------------------------------

## CORE RULES

-   Design deve ser moderno, elegante e diferenciado
-   Evitar aparência genérica
-   Foco em user experience e visual impact
-   Código clean, modular e reusable

------------------------------------------------------------------------

## TECHNOLOGY STACK (REQUIRED)

-   HTML
-   CSS
-   JavaScript

### Rules:

-   Usar modern best practices
-   Priorizar performance e scalability
-   Utilizar libraries leves quando necessário
-   A sidebar e o header devem obrigatoriamente estar dentro do container principal

------------------------------------------------------------------------

## PERFORMANCE & OPTIMIZATION (REQUIRED)

-   Deve ser lightweight e fast
-   Evitar heavy libraries
-   Priorizar CSS e JS nativo
-   Animações devem usar:
    -   transform
    -   opacity
-   Evitar:
    -   reflows
    -   lag

------------------------------------------------------------------------

## COLOR SYSTEM (DYNAMIC)

-   Sidebar e header devem seguir a cor definida pelo usuário
-   Criar:
    -   gradients
    -   glow effects

------------------------------------------------------------------------

## SIDEBAR (REQUIRED)

### Structure:

-   Posicionada à esquerda
-   Collapsed por padrão
-   Expand on hover

### Behavior:

-   Ao passar o mouse:
    -   expand smoothly
    -   revelar textos
    -   animar elementos

### Style:

-   Glassmorphism leve (se combinar)
-   Cores baseadas no tema
-   Gradients em formato de ondas:
    -   topo
    -   base

### Effects:

-   LED gradient border na direita
-   Neon glow sutil
-   Sombra visivel

### Icons:

-   Utilizar Lucide icons + Fake 3D (Obrigatório)
-   Ícones devem ter:
    -   hover animations
    -   scale
    -   glow
    -   micro-interactions

### Logo Section:

-   Parte superior deve conter:
    -   Logo enviada pelo usuário
    -   Texto: "Performance"

### Logo Effects:

-   Motion effects
-   Glow
-   Subtle animation

### Restrictions:

-   Não incluir:
    -   logout button
    -   notifications
    -   theme switch
    -   qualquer item não solicitado

------------------------------------------------------------------------

## HEADER (REQUIRED)

### Structure:

-   Topo da página
-   Full width

### Style:

-   Glassmorphism
-   Background quase transparente (10% de opacidade)
-   Blur (backdrop-filter)

### Content:

-   Left:
    -   "Dashboard"
    -   Nome da página
-   Right:
    -   Nome do usuário
    -   Foto de perfil

### Rules:

-   Apenas esses elementos
-   Layout limpo e equilibrado

------------------------------------------------------------------------

## EFFECT SYSTEM

-   Smooth transitions
-   Hover animations
-   Glow effects
-   Micro-interactions

------------------------------------------------------------------------

## AI DESIGN BEHAVIOR

-   Agir como senior designer
-   Melhorar automaticamente
-   Adicionar efeitos inteligentes

Sempre entregar um design moderno, fluido, otimizado e visualmente
impressionante
